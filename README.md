## Getting Started

BTCPay server credentials need to be provided. Create a secrets file.

```bash
touch .env.local
```

```
BTCPAY_URL=****
BTCPAY_AUTH=****
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
